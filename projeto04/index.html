Serviços AWS Utilizados

AWS ROUTE 53

O Amazon Route 53 é um serviço da Web de Sistema de Nomes de Domínio (DNS) altamente disponível e escalável. O Route 53 conecta as requisições do usuário a aplicações da Internet executadas na AWS ou on-premises.

AWS CERTIFICATE

Use o AWS Certificate Manager (ACM) para provisionar, gerenciar e implantar certificados SSL/TLS públicos e privados para uso com serviços da AWS e seus recursos internos conectados. O ACM elimina processos manuais demorados, como compra, upload e renovação de certificados SSL/TLS.

Amazon S3

Foi construído para armazenar e recuperar qualquer volume de dados de qualquer local, é um serviço de armazenamento de objetos que oferece escalabilidade, disponibilidade de dados, segurança e performance líderes do setor.

AWS CLOUD FRONT

O Amazon CloudFront é um serviço de rede de entrega de conteúdo (CDN) criado para alta performance, segurança e conveniência do desenvolvedor. Entregue conteúdo com segurança com baixa latência e altas velocidades de transferência.
